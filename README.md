# TaskFlow App 🗂️

TaskFlow is a full-stack task management web app built using Node.js, Express, MongoDB, and EJS. It features user authentication, CRUD operations, and personalized task tracking. Users can organize tasks by time and priority for better productivity.

## 🚀 Features

* User Authentication (Login & Signup)
* Create, Edit, Delete Tasks
* Time-based categorization (Morning/Afternoon/Night)
* Priority-based sorting
* Flash messages for feedback

## 🛠️ Tech Stack

* Node.js, Express.js
* MongoDB (Mongoose)
* EJS, CSS, JavaScript
* Passport.js, express-session


Open: http://localhost:8080
